# The 5 Lotus Psychotherapy & Research Center Website

This repository contains the full source code for the The 5 Lotus website,
including HTML pages, CSS, JavaScript, and assets.

## Pages Included
- index.html
- about.html
- services.html
- clinical-specializations.html
- psychological-tests.html
- contact.html
- team.html (if applicable)
- style.css
- script.js
- images/

## Instructions
1. Clone or download the repository.
2. Open index.html in a browser to view the site locally.
3. Edit any files in your favorite code editor.
4. Optional: Upload changes or create a pull request.
